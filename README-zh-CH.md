### 介绍

现在信息时代发展十分迅速，但是上班人群在上班途中过程十分拥挤，要搭顺风车网站是一款能够让挤公交车地铁的上班一族能够舒适出行，省时省力省钱的网站，还能减轻公共交通压力和节能环保。该网站不仅包含车主线路和乘客线路，还可以为车主提供拼车需求，又可为上班族提供更省心拼车方式，实现省钱、省时、舒适的拼车上下班生活。爱拼车，爱生活，爱上拼车生活。
